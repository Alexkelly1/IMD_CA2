My Projects

Second Year Project

In the second year of my degree I was part of a group project. The objective of the project was to use Django to design a website that people would use.

We made an app, similar to Just Eat and Deliveroo where users could order food from a local grocery store or delicatessan. We used the programming language Python to develop the website along with Django.

Users could select a store or deli and select the items that they wanted from there with a pre-prepared list we generated in a database. We also used Stripe within the project so the users could finish off by 'paying' for their products.

Fourth Year Project

In my current, and final, year I am developing a Transport Helper application using React Native in Javascript.

A user can register for the application and then, once registered, login to the application.

Once registered/logged in users can save their frequently visited locations right to their account for ease of use. A naming convention consisting of a dropdown list including the likes of “Home”, “Work”, “School” etc and “Custom”. With “Custom”, once users choose this, they can name a location whatever they want.

Once a start location and an end location are entered by the user, the application will then provide multiple transport modes from A to B. These modes consist of Car, Bus, Bike, Luas, Train.

The application compares each transport mode on Travel Time, Cost, Environmental Scale, Fuel Consumption, Fuel Cost, and Parking.

Users are those with a mobile phone which is connected to mobile date, concerned about how they travel and what it does for the environment and their time.
